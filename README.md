# 🛡️ Bank Fraud Detection & Financial Risk Analytics

> **An end-to-end data analytics and machine learning project for detecting fraudulent banking transactions using SQL, Python, Power BI, and Logistic Regression.**
##📖 About the Project

Financial fraud is one of the biggest challenges faced by banks and financial institutions. This project analyzes banking transaction data to uncover fraud patterns, build predictive models, and visualize insights through an interactive Power BI dashboard.

The project combines **Data Analysis**, **Business Intelligence**, and **Machine Learning** to provide a complete fraud detection workflow.

# 🎯 Objectives

✔ Clean and preprocess raw transaction data

✔ Perform Exploratory Data Analysis (EDA)

✔ Analyze transaction trends using SQL

✔ Build an interactive Power BI Dashboard

✔ Predict fraudulent transactions using Logistic Regression

✔ Evaluate model performance using classification metric
